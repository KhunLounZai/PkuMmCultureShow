# 视频文件目录结构说明

## 目录位置
```
public/images/videos/
```

## 文件命名规范
视频文件应按照以下格式命名：
```
{城市名称}.mp4
```

例如：
- `yangon.mp4` - 仰光城市视频
- `mandalay.mp4` - 曼德勒城市视频
- `bagan.mp4` - 蒲甘城市视频
- `inle.mp4` - 茵莱湖城市视频
- `ngapali.mp4` - 额布里海滩城市视频

## 视频要求
- **格式**: MP4
- **时长**: 8秒
- **分辨率**: 建议 1920x1080 或 1280x720
- **编码**: H.264
- **大小**: 建议每个文件不超过 10MB

## 使用说明
1. 将准备好的城市视频文件放入此目录
2. 确保文件名与城市数据中的 `slug` 字段匹配
3. 视频将在页面中自动播放，循环播放，无声音
4. 采用明信片样式展示，与图片展示区保持一致的设计风格

## 当前支持的城市
根据 `src/data/cities.json` 中的数据，当前支持以下城市的视频：
- yangon (仰光)
- mandalay (曼德勒) 
- bagan (蒲甘)
- inle (茵莱湖)
- ngapali (额布里海滩)

请将对应的视频文件放入此目录中。
